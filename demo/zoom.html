How it works

Set the focus-section attribute (or the focusedSection property) to a section name and the map zooms to it. Adjust the margin with focus-padding. Clear it to zoom back out.

The map scales to its container via the SVG viewBox, so resizing the frame (drag the corner, or use the size buttons) reflows smoothly with no re-layout.

ring-track-map Component Demo

ring-track-map — interactive SVG map of the Nürburgring Nordschleife. Renders the track from GPX data (via the gpx-url attribute or ) with the 21 named sections as clickable, hoverable segments. Sections can be highlighted, colored, and zoomed to.

Methods

MethodReturnsDescription
loadFromString(gpxText: string) => voidParse a GPX string and render the track. Useful for tests and inlined data.
highlightSection(sectionName: string) => voidSelect and highlight a single section by name.
highlightSections(sectionNames: string | string[]) => voidHighlight a set of sections (reflected in the highlight-sections attribute).
setSectionColor(sectionName: string, color: string) => voidAssign a custom highlight color to a section.
setSectionColors(colorMap: Record<string, string>) => voidAssign custom highlight colors for several sections at once.
clearSectionColors() => voidRemove all custom section colors.
focusSection(sectionName: string, padding?: number) => voidZoom the viewBox to a section's bounding box and select it.
resetView() => voidReset zoom, selection, and highlights to the initial full-track view.
